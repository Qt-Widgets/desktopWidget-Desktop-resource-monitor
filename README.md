# desktopWidget
Qt projects: Desktop Monitor Widget

:ru: Проект Qt 5 - Монитор ресурсов для рабочего стола

В центре карта мира и текущее положение.

На нижней панели монитор ресурсов:
- Загрузка процессора
- Память
- Температура
- Жесткие диски
- Сетевая активность
